# 写真管理 要件

## 概要
旅行内に写真を登録し、日程やスポットに紐付けて参照できる。

## 目的
- 旅の記録を視覚的に残せる
- ふりかえりや共有の材料になる

## 主要UI
- 写真ギャラリー
- Item/日付への紐付け
- 写真詳細モーダル

## 仕様
- 保存方式はクラウドストレージを前提とする
- 写真は旅行全体、日別、Item単位で紐付け可能
- 画像形式は JPEG/PNG/HEIC を想定
- 1枚あたり最大サイズを制限（例: 5MB）
- 画像は自動で縮小・圧縮してアップロード
- 画像にキャプションとタグを付けられる
- 権限に応じて閲覧/アップロード可否を制御する

## 受け入れ基準
- 写真を追加するとギャラリーに表示される
- 紐付け先（旅行/日/Item）で絞り込みできる
- キャプションを編集できる
- 権限がないユーザーはアップロード不可

## エッジケース
- 大容量画像のアップロード時に警告を表示
- アップロード失敗時はリトライ導線を表示
- ネットワーク不安定時はアップロード状態を表示
