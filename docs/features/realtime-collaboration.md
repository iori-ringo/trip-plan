# リアルタイム共同編集 要件

## 概要
旅行計画を複数人で同時に更新できる。

## 目的
- 友人と計画を一緒に作る際の往復を減らす
- 編集内容が即時に反映される

## 主要UI
- 共有リンク生成
- 参加者リスト
- 変更履歴/最近の更新表示

## 仕様
- 共同編集はログイン済みユーザーのみ利用可能
- 旅行ごとに共有リンクを発行できる
- 参加者は「閲覧のみ / 編集可能」を選択可能
- 同期方式はWebSocketを利用する
- 編集内容は自動で同期される
- 同時編集の衝突は「最後の更新を採用（Last Write Wins）」を基本
- 変更履歴を直近20件まで表示

## 受け入れ基準
- 2人以上で同じ旅行を開き、変更が即時反映される
- 閲覧権限は編集できない
- 未ログイン状態では共同編集画面に入れない
- 変更履歴が更新される

## エッジケース
- オフライン復帰時に競合が発生した場合は通知
- 共有リンクの無効化ができる
- 共有リンク失効後は再接続できない
