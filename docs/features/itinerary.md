# 日程タイムライン 要件

## 概要
日付ごとにItemを並べて旅程を組み立てる。

## 目的
- 1日の流れを時間軸で把握できる
- 並び替えや時間調整を素早く行える

## 主要UI
- 日付タブ
- タイムラインリスト
- 未割当エリア

## 仕様
- 日付は旅行期間から自動生成される
- Itemは開始時間順に並ぶ
- 開始時間が未設定のItemは日付内の末尾にまとめる
- ドラッグ&ドロップで並び替えできる
- 日付内にメモを保存できる

## 受け入れ基準
- 日付タブの切り替えで対象日程が切り替わる
- 並び替えが直感的に行える
- 未割当から各日へItemを移動できる
- 1日単位のメモが保存される

## エッジケース
- 開始時間が同一の場合は追加順で安定的に並ぶ
- 時間未設定のItemが多数でも操作が破綻しない
