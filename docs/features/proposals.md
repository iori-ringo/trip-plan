# 機能提案（ペルソナ課題の解決案）
作成日: 2026-02-05

## MVP候補
- 未割当ボックス
- 確定していない候補を一時保管し、日程へドラッグで移動
- 日程タイムライン
- 時間順の並び替え、時間の簡易編集
- スポットカード
- 地図リンク、メモ、費用メモを1画面で確認
- 日別メモ
- その日の注意点や買い物メモを保存
- ローカル保存
- オフライン閲覧の前提を作る
- JSONエクスポート/インポート
- バックアップと移行を可能にする

## Next候補
- 閲覧専用の共有モード
- 編集権限を渡さずに共有できる
- 簡易予算集計
- 日別/カテゴリ別の支出メモ合計
- 割り勘
- 立替の記録と精算結果を自動算出
- チェックリスト
- 持ち物や準備の確認
- 旅程の印刷用ビュー
- 1枚で確認できる要約ページ
- 写真管理
- 旅行/日程/Itemに紐付け、ふりかえりに活用

## Stretch候補
- リアルタイム共同編集（ログイン必須 + WebSocket）
- 友人と同時編集し、変更が即時に同期される
- 予約メールの取り込み
- 手入力負担を下げる
- 地図の埋め込みビュー
- 行程の俯瞰確認
- AIによる行程提案
- 候補地の並びや時間配分の提案
