# 旅行のふりかえり 要件

## 概要
旅行終了後に、日程や写真、メモを整理して振り返れる。

## 目的
- 旅行の記録を残し、次回の参考にできる
- 共有したい思い出を素早くまとめられる

## 主要UI
- 旅行サマリービュー
- 日別サマリー
- ハイライト/お気に入り

## 仕様
- ふりかえりは旅行終了後のみ表示する
- 日別に「良かった点」「注意点」「次回やりたいこと」を記録できる
- Item単位でハイライト（お気に入り）を付けられる
- サマリーをMarkdownでエクスポートできる

## 受け入れ基準
- 旅行終了後にのみサマリーが表示される
- 日別メモが保存される
- ハイライトが一覧に反映される
- Markdownエクスポートができる

## エッジケース
- 旅行途中はふりかえりにアクセスできない
- ふりかえり未入力でもエクスポート可能
